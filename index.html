PureMDread

A clean, local Markdown reader Chrome extension
Beautiful rendering, no conversion Light/Dark themes Syntax highlighting Auto TOC

Features

Core Rendering

  • Auto-detect Render - Automatically renders .md / .markdown / .mdown / .mkd
  • GitHub Style - Faithfully reproduces GitHub Flavored Markdown
  • GFM Support - Tables, task lists, strikethrough, etc.
  • Syntax Highlighting - Built-in Highlight.js, all languages

Theme System

  • Light Dark - One-click toggle
  • Preference Memory - Auto-saved and applied
  • Eye-Friendly - Soft #e0e0e0 gray background

Reading Experience

  • Smart TOC - Auto-generated clickable table of contents
  • Font Size - A- / Reset / A+, 12px-24px range
  • Print/PDF - One-click export
  • Image Path Fix - Auto-resolve relative paths

Performance

  • Zero-latency - document_start injection
  • No Network - Runs entirely locally
  • Lightweight - 500KB package

Installation

Developer Mode (Recommended)

  1. Download repository ZIP or clone locally
  2. Open Chrome, navigate to chrome://extensions/
  3. Enable Developer mode
  4. Click Load unpacked
  5. Select markdown-reader-extension folder
  6. Important: Extension Details Enable Allow access to file URLs
Coming soon to Chrome Web Store

Usage

  1. Drag a local .md file into Chrome
  2. Or enter file:///path/to/file.md in the address bar
  3. The extension auto-renders for a beautiful reading experience
